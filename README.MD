如何写一段完美的递归函数？

1. 确定该任务是否可分？（比如常见的反转链表、计算n次幂、机器人移动范围等等）
2. 确定该任务的终止条件。（写在recursion函数的第一排）
3. 如果没有到中止条件，则进行递归计算
    1. 首先写递归函数进行迭代
    2. 然后写递归的计算部分
